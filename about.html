Sobre Nós

Conheça mais sobre nosso projeto

O Catálogo GameCatalog

Missão

Nosso objetivo é criar um espaço simples e acessível para descobrir e conhecer mais sobre os melhores jogos do mercado. Acreditamos que a tecnologia deve ser simples, clara e fácil de entender.

Tecnologia

Este projeto foi desenvolvido utilizando apenas:

  • HTML5 - Estrutura semântica e acessível
  • CSS3 - Estilos minimalistas e responsivos
  • JavaScript - Interatividade simples e funcional

Sem frameworks ou bibliotecas externas, focando em conceitos fundamentais que qualquer estudante de programação pode aprender e modificar.

Aprendizado

Este catálogo é um ótimo exemplo para aprender:

  • Estrutura básica de páginas web
  • Navegação entre páginas
  • Passagem de parâmetros via URL
  • Manipulação do DOM com JavaScript
  • Responsividade com CSS Grid e Flexbox
  • Boas práticas de código limpo e organizado
